```bash

npm install

# 2. Start
```

## 🐛 Issues?

### Backend not running?
```bash
# In project root
python orchestrator/main.py
```

### npm install fails?
```bash
npm cache clean --force
npm install
```

### Port 3000 busy?
Edit `vite.config.ts` and change port.
